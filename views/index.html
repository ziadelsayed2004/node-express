Welcome to My API

Ziad Elsayed – Backend Developer using Node.js and Express. This is a demo API project that includes user authentication, course management, and file uploads.

/api/users

  • GET /api/users – Requires token – Get all users
  • POST /api/users/register – Public – Register a user with image upload (avatar)
  • POST /api/users/login – Public – Login and receive a JWT token

/api/courses

  • GET /api/courses – Public – Fetch all courses
  • POST /api/courses – Requires token (Manager role) – Add a new course
  • GET /api/courses/:courseId – Public – Get specific course by ID
  • PATCH /api/courses/:courseId – Requires token (Admin or Manager) – Update course
  • DELETE /api/courses/:courseId – Requires token (Admin or Manager) – Delete course

Static & Uploads

  • GET / – Main documentation page
  • GET /home – Redirects to `/`
  • 404 Page – Displayed when route is not found
  • /uploads/filename – Access uploaded images directly via this path

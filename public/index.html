/api/users

  • GET /api/users – Requires token – Get all users
  • POST /api/users/register – Public if role is USER, Requires token if role is ADMIN or MANAGER – Register a user with avatar image upload
  • POST /api/users/login – Public – Login and receive a JWT token

/api/courses

  • GET /api/courses – Public – Fetch all courses
  • POST /api/courses – Requires token (ADMIN or MANAGER) – Add a new course
  • GET /api/courses/:courseId – Public – Get course by ID
  • PATCH /api/courses/:courseId – Requires token (ADMIN or MANAGER) – Update a course
  • DELETE /api/courses/:courseId – Requires token (ADMIN or MANAGER) – Delete a course
  • POST /api/courses/enroll/:courseId – Requires token (USER) – Enroll in a course
  • DELETE /api/courses/unenroll/:courseId – Requires token (USER) – Unenroll from a course
  • GET /api/courses/my-courses – Requires token (USER) – Get your enrolled courses
  • GET /api/courses/my-teaching-courses – Requires token (ADMIN) – Get courses you're teaching
  • GET /api/courses/users-courses/:userId – Requires token (MANAGER) – Manager can view a user's courses

Static & Uploads

  • GET / – Main documentation page
  • GET /home – Redirects to `/`
  • 404 Page – Displayed when route is not found
  • Get Uoloded /uploads/filename – Access uploaded images directly via this path
